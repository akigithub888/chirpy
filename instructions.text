to build and run as server

go build -o out && ./out


./out is just to run it

go run .      works as well

Each time you change your code you'll need to rebuild and restart your server

starting postgres server:

sudo service postgresql start

to enter postgres shell

sudo -u postgres psql 

CONNECTION STRING:

psql "postgres://aleksa:akilukidakimaki@localhost:5432/chirpy"

MIGRATIONS:

goose postgres "postgres://aleksa:akilukidakimaki@localhost:5432/chirpy" UP / DOWN
